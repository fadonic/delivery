# Delivery Application

Delivery application is a project that provider delivery services to both individual and organisation.

## Available Modules

The project consist of:

* Client
* Server

### `To configure and run the project`
```
cd server
npm install
run npm start
```

```
cd client
npm install
run npm start
```
## Build With

### Client
* React
* React Router
* React Hooks
* Context API
### server
* Node
* Express
* JWT Authentication
* Mongo Db
